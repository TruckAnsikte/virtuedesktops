About the Desktops Inspector
The Desktops Inspector is Virtues interface to organize your collection of Virtual Desktops. It manages the heap of available Virtual Desktops in an unordered manner and provides functionality to add new Virtual Desktops to the pool and remove existing Virual Desktops from the pool.

To open the Desktops Inspector,

  • Open the Virtue status bar menu
  • Select Desktops > Inspect Desktops... from the opened menu
