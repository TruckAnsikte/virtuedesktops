About the Desktop Inspector
The Desktop Inspector is the primary interface for managing the pool of available Virtual Desktops and for customizing the appearance of each Virtual Desktop. You can use the Desktop Inspector to change the wallpaper of a Virtual Desktop or to create a new Virtual Desktop.

To open the Desktop Inspector for the currently visible Virtual Desktop,

  • Open the Virtue status bar menu
  • Select Active Desktop > Inspect Active Desktop... from the opened menu
