Creating a new Trigger
Triggers are bound to Actions in the Triggers Preferences. To open the Triggers Preferences,
  • Select Virtue > Preferences...
  • Highlight the Triggers entry
Remember, you can bind multiple Triggers to an Action, but a Trigger may only bound to a single Action. To create a new Trigger,
  • Press the Add button
  • Highlight the Action you want to bind int the Panel that opened and press the Select button
  • Push the Trigger Type icon in the list entry of the new Trigger to select the type of the Trigger
  • Push the Info button to assign modifiers and key / edge to the Trigger
