Using Screen Edges
Switching Virtual Desktops via screen edges is making use of Virtues mouse trigger action bindings.

As with all other mouse triggers, you can assign modifier keys, delays and mandatory click counts to screen edges. To bind a new screen edge to switch to another Virtual Desktop,

  • Open the Triggers preferences by selecting Virtue > Preferences > Triggers
  • Push the Add Trigger button to create a new Action Binding
  • Select the Navigation > Show desktop to the north action (or any other direction) and push the Select button
  • Highlight the new Binding and change the type to Mouse Trigger
  • Select the upper edge and optionally set modifiers and delay
