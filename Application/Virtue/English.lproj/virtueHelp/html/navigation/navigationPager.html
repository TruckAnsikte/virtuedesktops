Using the Pager
The pager provides an easy and efficient way of navigating between your Virtual Desktops. It can be opened in two different modes; the sticky-mode and the normal-mode. Once opened, you can use either the mouse or the keyboard to highlight and select a Virtual Desktop to show it.

To highlight desktops via the keyboard use the arrow keys to select an adjacent desktop. To show the currently selected desktop, use either the Enter or Space key. You can dismiss the pager by either clicking outside the pager or by pressing the Escape key.
