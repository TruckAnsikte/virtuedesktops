Naming your Virtual Desktops
Each Virtual Desktop managed by Virtue has to have a unique name. New desktops created in the Desktops Inspector will be assigned a unique default name, your are free to modify. To assign names to Virtual Desktops, open the Desktop Inspector for the Virtual Desktop you want to name,
  • Open the Desktop Inspector
  • Highlight the Virtual Desktop you want to name
  • Enter the new name in the Name text field and press the Enter key to commit to the name
Alternatively you can directly change the name in the available desktops list by double-clicking a desktop entry there.
