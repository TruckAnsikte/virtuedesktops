Organizing your Virtual Desktops
Virtue is configurable in the number of Virtual Desktops it provides to you at a time. You can create new Virtual Desktops and remove existing ones easily. To see the pool of existing Virtual Desktops Virtue provides the Desktop Inspector. To add a new Virtual Desktop and name it immediately,
  • Open the Desktop Inspector
  • Push the Add button
  • Highlight the new Virtual Desktop to give it a reasonable name and configure its appearance
Note that when adding new Virtual Desktops they will be given a default name, that is merely useful for anything but finding it again in the list to set it up to your likening.

Removing a Virtual Desktop is similarly easy,

  • Open the Desktop Inspector
  • Highlight the Virtual Desktop you want to remove
  • Push the Delete button
