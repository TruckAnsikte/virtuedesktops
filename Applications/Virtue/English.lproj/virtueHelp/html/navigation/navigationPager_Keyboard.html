Using the Pager
With the pager open, you can use the keyboard arrow keys to highlight desktops adjacent to the currently highlighted desktop. By pressing the Space or Enter key, you show the highlighted desktop. The pager can be dismissed by pressing the Escape key.

When hitting an edge in the pager layout, the pager can either jump to the next row and highlight the first desktop in the new row, or simply jump to the opposite end in the same row. To configure the behaviour when hitting the edge of a row,

  • Open the pager preferences by selecting Virtue > Preferences > Layout & Pagers
  • Check the Stay in current row option to not jump to another row; or Uncheck the Stay in current row option to jump to another row when hitting an edge
If your desktop layout contains empty desktop cells between two desktops, the pager will normally jump over this empty cell. If you do not want to skip empty cells to emphasize the intentional placing of the empty cell in a row,
  • Open the pager preferences by selecting Virtue > Preferences > Layout & Pagers
  • Uncheck the Skip empty desktop cells option
