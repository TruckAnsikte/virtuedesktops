Using Hotkeys
Switching Virtual Desktops via hotkeys is making use of Virtues keyboard trigger action bindings.

As with all other keyboard triggers, you can assign modifier keys to the action binding used to switch between desktops. To bind a new hotkey to switch between desktops,

  • Open the Triggers preferences by selecting Virtue > Preferences > Triggers
  • Push the Add Trigger button to create a new Action Binding
  • Select the Navigation > Show desktop to the north action (or any other direction) and push the Select button
  • Highlight the new Binding
  • Double click the Trigger field and press the hotkey combination that should be used to activate the action
