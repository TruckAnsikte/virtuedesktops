About Triggers
Triggers allow you to bind Actions you want to access frequently to a convinient shortcut so you can reach them easily and be efficient.

Virtue provides a wide range of Actions that can be bound to either Keyboard shortcuts or Mouse shortcuts called Keyboard Triggers and Mouse Triggers. An Action provided by Virtue can have multiple Triggers associated with it, but a Trigger can only be assigned to one Action at a time.

Triggers can be added and removed from within Virtues preferences. To open the Triggers Preferences,

  • Select Virtue > Preferences...
  • Highlight the Triggers entry in the list of preference panes
