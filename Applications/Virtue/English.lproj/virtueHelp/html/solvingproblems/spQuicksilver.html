Why do Quicksilvers mouse-triggers only work on one desktop?
Quicksilvers mouse triggers will only work on one desktop in their current implementation (B32r2). If you want to access Quicksilvers mouse triggers on all your Virtual Desktops, you will have to sticky the Quicksilver application.

To sticky the Quicksilver application (or any other application),

  • Start Quicksilver if it is not running
  • Open the Application Inspector by selecting the Virtue menu > Inspection > Applications...
  • Highlight the Quicksilver application in the list of applications
  • Check the Show on all desktops option
Quicksilvers Mouse-Triggers will now work on all Virtual Desktops. Virtue will remember this setting through restarts, so you do not have to repeat these steps once Virtue or Quicksilver are restarted.
