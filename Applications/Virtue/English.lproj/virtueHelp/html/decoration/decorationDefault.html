Defining a default Desktop Decoration
Virtue defines an empty Desktop Decoration per default for each new Virtual Desktop you create. To define a default Desktop Decoration that is assigned to all new Virtual Desktops,
  • Open the preferences by selecting Virtue > Preferences
  • Highlight the Desktop Template pane
  • Add new Decoration Primitives by pressing the Add Primitive button and configure them by pressing the Info button.
If you want your template to be assigned to all new desktops, check the Use Template for new desktops option. Virtue will now assign your template to new desktops.

If you want to immediately apply the template to all of your existing Virtual Desktops,

  • Open the preferences by selecting Virtue > Preferences > Desktop Template
  • Press the Replace Now button to assign the template to all existing desktps
