Adding a Decoration Primitive
Decoration Primitives are managed by the Desktop Inspector. To add a new decoration primitive,
  • Open the Desktop Inspector and highlight the desktop you want to decorate
  • Push the Add primitive button located below the list of primitives
  • Select the type of primitive to add in the opened menu
The primitive is now added to the decoration and can be configured by opening the Primitive Inspector Panel by pressing the Info button.

Decorations display their primitives from the top of the list to the bottom, you can reorder them by dragging primitives around.
