What is new in Virtue
Virtue 0.5r2 is the second release in the Virtue 0.5 line, providing bug fixes and new features as reported by you, the precious user.
  • Be efficient with the keyboard
    The buit-in pager was enhanced to allow more keyboard navigation. Use the arrow keys to navigate, return and space to select and the escape key to abort. On popup the shown desktop is selected, so you can start browsing right away without finding your dusty mouse.
  • Lay out your desktops to match your work-style
    The pager now controls how your desktops are ordered. Create your virtual desktops in the Desktops Inspector and order them to your likening in the pager via Drag and Drop, it is that easy!
  • More support for non-Finder users
    Virtue is now no longer depending on a running Finder, but is also working with alternatives like Path Finder, and wll not insist on restarting the Finder. If you don't like it, Virtue is not forcing you.
  • Extended Triggers
    You are now no longer dependant on the keyboard to reach various functions Virtue provides. Triggers allow you to bind actions to either hotkeys or edges of the screen. Open the pager via the mouse by throwing the mouse pointer to the left edge of the screen and click twice there for example. Its fun, try it.
