Using the Desktop Pager

A pager is an on-screen tool delivered with most virtual desktop managers that allows you to easily navigate between your desktops.

Virtue's pager is only shown when you need to use it, and is designed for quick operation while still providing a good overview of available desktops. The pager can be operated in two distinct modes:

  • Normal Mode
    In normal mode, the pager will display only as while the Trigger Modifiers used to bring it up are pressed.
  • Sticky Mode
    In sticky mode, the pager will stay on the screen as long as it has focus and you don't select another desktop. Once the pager loses focus (e.g. another window is activated), it will close.

To navigate your desktops using the pager:

You can highlight desktops via the keyboard, or via the mouse.

  1. First, bring the pager up - we'll do this using the stick mode for this example by pressing Shift + Tab (this can be customised later).
  2. If you're using your mouse to select desktops simply hover over the desktop you wish to switch to. If you're using your keyboard, use the arrow keys.
  3. To switch to the currently selected desktop, you can either click your mouse or press either the Enter or Space key.

You can also dismiss the pager by either clicking outside the pager window, or by pressing the Escape key.
